# はいはい！ベビーラン — 確定仕様（初期値）

> プロトタイプ右下の **Tweaks → 「📋 仕様書を出力」** で、その場で調整した数値を反映した最新版 Markdown を生成・コピーできます。これはデフォルト値のスナップショットです。Claude Code 実装用。

## スピード
- スクロール速度: **200 px/s**
- 赤ちゃん横移動速度: **440 px/s**
- px→m 換算: **38 px = 1 m**

## 難易度（スポーン）
- スポーン間隔: **0.75 s**
- 出現比率 — 障害物 **45%** / おもちゃ **30%** / 回復アイテム **25%**
- 回復アイテム内訳 — 哺乳瓶 **60%** / オムツ **40%**
- 配置: 5レーンにランダム、サイズは個体ごとに **0.78〜1.33倍** でばらつき。動的(ロボット/ボール)は **斜め方向**へ移動し側壁で反射、静的(本/テディ/アヒル/哺乳瓶/オムツ)は背景と同速。
- オブジェクト構成 — 障害物: **ロボット(動的)・本(静的)** / おもちゃ: **ボール(動的)・テディベア(静的)・アヒル(静的)** / 回復: **哺乳瓶・オムツ**
- 時間経過での難易度上昇なし（一定テンポのリラックス系）。

## パラメータ挙動
- 初期体力 / 最大体力: **100**
- 体力消費: 距離 **0.35/m**（不快度 80% 以上で **×2**）
- 障害物ダメージ: **-10** / おもちゃダメージ: **-10**
- 哺乳瓶回復: 最大体力の **20%**（上限100まで）
- オムツ: 不快度を **0%** にリセット
- 不快度: 0% → 100% を **30s** で自動上昇（難易度非依存）、100%で「！パンパン！」表示
- 無敵時間: 衝突後 **1.0s**（点滅）
- 接触フリーズ: **0.6s** 間スコア・距離の加算を停止
- 体力 0 でゲームオーバー

## アニメーション（要確定 — プロトタイプで案を切替）
- ハイハイ: 既定 **A 対角**（左腕と右脚が交互に振れ、上下に小さく弾む）
  - 候補: A 対角 / B うさぎ跳ね（両腕→両脚） / C くねくね（左右に体を振る）
  - サイクル: **2.2/s**（スクロール速度に比例）、上下バウンド **7px**
- 被弾演出: 既定 **A 点滅＋赤グロー＋色相シフト＋小刻みジッター**
  - 候補: A 点滅 / B 回転（転ぶ） / C 潰れ（縦に押し潰れる）
- 遊ぶ演出: 既定 **A 揺れ**（座って左右に揺れる）
  - 候補: A 揺れ / B 跳ね / C 回転（一回転）
- 画面シェイク（被弾時）: 強さ **8px** / 減衰 **0.3s**
- 不快度80%以上: 困り顔＋汗の表情に切替
- 接触テキスト: 障害物=「いたっ！」/ おもちゃ=「わーい！」を赤ちゃん頭上に大きくスケールイン表示
- ポップテキスト: 「+体力」(緑) /「おむつ交換！」(青) /「いたっ！」(赤) /「遊んじゃった！」(橙)、上方向にフェードアウト
