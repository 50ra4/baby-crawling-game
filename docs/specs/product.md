# プロダクト仕様（product）

> はいはい！ベビーラン（baby-crawling-game）のプロダクト要件・ユーザーストーリー・用語集。
> 本書は**実装を正本**とする。挙動・数値は `src/game/`・`src/constants/gameConfig.ts`・`src/components/screens/` を参照。
> 関連: [tech.md](./tech.md) / [structure.md](./structure.md) / [DESIGN.md](./DESIGN.md)

## 1. コンセプト

赤ちゃんがハイハイで進む距離（m）を競う、縦スクロール型の癒し系無限ランナー。
プレイヤーは赤ちゃんを左右に動かし、おもちゃで遊んだり、一定間隔で流れてくる哺乳瓶・オムツで
体力／不快度（おむつ）を管理し、どこまで遠くへ進めるかに挑戦する。絵本トーンの淡い配色と
丸ゴシックでまとめた、短時間で気軽に遊べるカジュアルゲーム。

- プラットフォーム: モバイル／PC ブラウザ（GitHub Pages で配信）
- 1 プレイ: 体力が尽きるまでのワンショット。スコア＝到達距離（m）
- 永続化: ベスト距離・名前・性別を localStorage に保存

## 2. ユーザーストーリー

- プレイヤーとして、名前と性別を設定し、自分の赤ちゃんとして遊びたい。
- プレイヤーとして、タップ／ドラッグでもキーでも直感的に左右移動したい。
- プレイヤーとして、体力と「おむつ（不快度）」を見ながら回復アイテムを取りに行く駆け引きを楽しみたい。
- プレイヤーとして、おもちゃに触れても即終了ではなく「遊んじゃった！」の演出で和みたい。
- プレイヤーとして、ベスト距離を更新し続けるモチベーションが欲しい。
- 初見プレイヤーとして、「あそびかた」でルール（操作・ステータス・おもちゃ・アイテム）を確認したい。

## 3. 画面フロー

`src/App.tsx` が画面状態 `GameScreen = 'title' | 'playing' | 'over'` を管理する。

```
タイトル ──[はじめる/Space/Enter]──▶ プレイ中 ──[体力0]──▶ ゲームオーバー
   │  ▲                                                        │   │
   │  └──────────────[あそびかたダイアログ]                    │   │
   │                                                            │   │
   └──◀──────────────[タイトルへ]◀────────────────────────────┘   │
                       [もういっかい]──────────────────────────────┘
```

| 画面           | 実装                | 主な要素                                                                                                                             |
| -------------- | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| タイトル       | `TitleScreen`       | ロゴ／ハイハイプレビュー／名前入力（最大 8 字）／性別（男の子・女の子）／「はじめる」／「あそびかたを みる」／操作ヒント／ベスト記録 |
| あそびかた     | `HelpDialog`        | そうさ・ステータス・スコア・きをつけて（おもちゃ）・アイテム。モーダル表示中は背景を `inert` 化                                     |
| プレイ中       | `Playfield` + `Hud` | 背景（テーマ別）／落下オブジェクト／赤ちゃん／HUD／各種演出                                                                          |
| ゲームオーバー | `GameOverScreen`    | 「ゲームオーバー」／「すやすや… おやすみなさい」／眠る赤ちゃん＋ZZZ／こんかい・ベスト／「もういっかい」「タイトル」                  |

ゲームオーバーは体力 0 検知から `GAME_OVER_DELAY_MS = 350ms` 後に `over` 画面へ遷移する。

## 4. コアゲームループ

`requestAnimationFrame` ループ（`useGameLoop`）が毎フレーム純粋関数 `stepGame` を呼ぶ。
`dt` は `MAX_DT = 0.05`（50ms）で上限クランプ。1 フレームの処理順は以下。

1. ハイハイ位相 `phase` を更新（スクロール速度に比例）
2. 接触フリーズ状態の更新（`updateContact`）
3. 赤ちゃんの横移動（`moveBaby`、**フリーズ中も継続**）
4. フリーズ中でなければ: 距離加算・スコア算出・オブジェクト移動（`moveObjects`）・スポーン（おもちゃ／哺乳瓶／オムツを独立間隔で `spawnObject`）
5. 体力消費・不快度上昇（`nextStamina` / `nextDiscomfort`、**フリーズ中も継続**）
6. 衝突判定と回復・遊び処理（`checkCollisions`）
7. ポップアップ寿命更新（`updatePopups`）
8. 体力 ≤ 0 でゲームオーバーイベント発火

> **接触フリーズ**中は「スコア・距離の加算」「スクロール」「スポーン」を停止するが、
> 横移動・体力消費・不快度上昇・アイテム/接触判定は継続する。

## 5. ゲームメカニクス（既定値＝normal プリセット）

数値は `DEFAULT_CONFIG`（`src/constants/gameConfig.ts`）が正本。

### 5.1 距離・スコア

- スクロール速度 `scrollSpeed = 200 px/s`
- 換算 `PX_PER_M = 38`（38px = 1m）
- スコア = `floor(distancePx / 38)`、距離（m）がそのままスコア

### 5.2 体力（stamina）

- 初期・最大 `staminaStart = 100`
- 消費 `drainPerSec = 2 /s`（時間ベース）
- 不快度が `discomfortThreshold = 80%` 以上で消費 `drainMultiplier = 2` 倍（= 4/s）
- 哺乳瓶で `bottleHealPct = 20%`（最大体力比）回復
- 体力 ≤ 0 でゲームオーバー

### 5.3 不快度（discomfort、UI 表記「おむつ」）

- 体力消費の 2 倍速（= 4/s）で 0→100% へ自動上昇（上限 100）
- オムツで 0% にリセット
- 100% で HUD に「！パンパン！」警告。80% 以上で赤ちゃんが困り顔表現に切替

### 5.4 スポーン

- おもちゃ・哺乳瓶・オムツを**種類ごとに独立した固定間隔**で出現させる（時間アキュムレータ方式、`Math.random` 非依存のタイミング）
  - おもちゃ `toyInterval = 1.2s`（種類は ball/teddy/duck から等確率抽選）
  - 哺乳瓶 `bottleInterval = 4.5s`（確実に供給）
  - オムツ `diaperInterval = 7.0s`（確実に供給）
- 出現位置 `y = -44`（画面上端外）、5 レーンからランダム
- 動的オブジェクト（ボール）のみ横初速 `±scrollSpeed × (0.28〜0.58)`

### 5.5 オブジェクト一覧（`OBJECT_META`）

| kind   | ラベル           | カテゴリ | 動的 | 基本サイズ | 接触効果                            |
| ------ | ---------------- | -------- | ---- | ---------- | ----------------------------------- |
| ball   | ボール           | toy      | ○    | 54         | 体力 −8「遊んじゃった！」play 演出   |
| teddy  | テディベア       | toy      | ×    | 76         | 体力 −8「遊んじゃった！」play 演出   |
| duck   | アヒルのおもちゃ | toy      | ×    | 42         | 体力 −8「遊んじゃった！」play 演出   |
| bottle | 哺乳瓶           | item     | ×    | 54         | 体力 +20%「+体力」                  |
| diaper | オムツ           | item     | ×    | 54         | 不快度 0%「おむつ交換！」            |

- 動的オブジェクトは落下速度 `scrollSpeed × 1.5`（= 300px/s）、側壁（内側 26px）で反射
- 静的オブジェクトは落下速度 `scrollSpeed`

### 5.6 衝突・無敵・接触

- 当たり判定: `|o.x − babyX| < r + 24 && |o.y − 476| < r + 26`（`r = base × scale × 0.42`）
- 回復アイテムは無敵に関係なく即時取得
- おもちゃに触れた後 `invincibleTime = 1.0s` 無敵
- 接触フリーズ `contactTime = 0.6s`

## 6. 操作

`src/hooks/useInput.ts` が集約。

- **キーボード**: ← / A で左、→ / D で右（押している間だけ移動）。Space / Enter で開始・再開
- **ポインタ**: タップ／ドラッグで指の論理 X 位置へ追従。指を離しても目標は保持され、そこへ移動を続ける（タップ移動）
- キー操作を始めるとタップ目標はクリアされ、キー入力が優先される
- 横移動は `babyMoveSpeed = 293 px/s`・`babyAccel = 1500 px/s²` でなめらかに加減速（急発進・急停止を避ける「遊び」）
- テキスト入力中・IME 変換中・ボタンフォーカス中・モーダル表示中は開始/移動キーを無視する

## 7. 永続化（localStorage）

`src/utils/storage.ts` が正本。破損・未保存時は安全な既定値へフォールバック。

| キー                | 内容                               | 既定     |
| ------------------- | ---------------------------------- | -------- |
| `baby_crawl_best`   | ベスト記録 `{dist, score}`（JSON） | `{0, 0}` |
| `baby_crawl_name`   | 赤ちゃんの名前                     | `''`     |
| `baby_crawl_gender` | 性別 `'boy' \| 'girl'`             | `'girl'` |

表示名は `displayName(name, gender)`（`src/utils/displayName.ts`）で組み立てる。
名前が空なら「あかちゃん」、それ以外は性別の敬称（boy=くん / girl=ちゃん）を付与する。

## 8. 用語集

| 用語                         | 意味                                                                            |
| ---------------------------- | ------------------------------------------------------------------------------- |
| 論理キャンバス               | 全座標計算の基準空間 360×680px（9:17 縦長）。実画面へはスケール＋レターボックス |
| レーン                       | オブジェクト出現位置の縦列。5 本、X = [48, 114, 180, 246, 312]                  |
| ハイハイ位相（phase）        | 0..1 を循環するアニメーション進行度。スクロール速度に比例して進む               |
| 体力（stamina）              | 0 で終了するメインゲージ。時間で減少、哺乳瓶で回復                              |
| 不快度（discomfort）／おむつ | 時間で上昇するサブゲージ。80% 以上で体力消費が加速、オムツで 0% に              |
| 接触フリーズ                 | 遊び時に距離加算・スクロール・スポーンを止める一時停止（0.6s）                  |
| 無敵時間                     | おもちゃに触れた後、再接触しない猶予（1.0s）                                    |
| 動的オブジェクト             | 横初速を持ち側壁で反射し、速めに落下するオブジェクト（ボールのみ）              |
| ポップテキスト               | 取得・遊び時に上昇フェードする数値/文言演出                                     |
| 接触バースト                 | 遊び時に赤ちゃん頭上へ出る大きい文言（「わーい！」）                            |
